# Training Framework

## Main Priorities

1. Gravel cycling performance
2. Strength and hypertrophy
3. Calisthenics skills
4. Mobility
5. Optional running/swimming for recovery or maintenance

## Weekly Structure Template

| Day | Priority |
|---|---|
| Monday | Gym A + short calisthenics push/handstand skill |
| Tuesday | Cycling VO2max / hill repeats |
| Wednesday | Gym B + calisthenics pull skill |
| Thursday | Cycling threshold / sweet spot / tempo |
| Friday | Gym C + short support/core skill |
| Saturday | Long gravel ride |
| Sunday | Recovery, mobility, easy spin/swim/walk, optional easy run |

## Cycling Training Types

### VO2max / Hill Repeats

Purpose:
- Improve ability to handle steep climbs, surges, and repeated hard gravel efforts.

Examples:
- 5 × 3 min hard / 3 min easy
- 6 × 4 min hard / 4 min easy
- 8–10 × 60–90 sec uphill hard
- 3 × 10 min blocks of 30 sec hard / 90 sec moderate

### Threshold / Sweet Spot

Purpose:
- Improve sustained gravel pace and climbing power.

Examples:
- 3 × 12–15 min sweet spot
- 4 × 8 min threshold
- 2 × 20 min controlled threshold
- 4 × 8 min low-cadence strength endurance at 60–70 rpm

### Long Gravel Ride

Purpose:
- Aerobic durability, fat oxidation, technical handling, fueling practice.

Default:
- 3–5 hours mostly Z1–Z2

Occasional additions:
- Tempo finish
- Climb blocks
- Rough terrain
- Race fueling practice
- Gravel handling drills

## Running

Running is optional support only.

Recommended:
- 0–1 easy run per week
- 20–45 min Z1–Z2
- No regular running VO2max
- No regular running threshold
- No weekly long run unless specifically maintaining running

## Swimming

Swimming is optional recovery/aerobic support.

Recommended:
- 0–2 easy swims per week
- 20–40 min technique or recovery
- Avoid hard swim sessions unless there is a specific reason

## Gym Training

Keep 3 gym days per week.

Current split:
- Day A: squat / horizontal push / vertical pull
- Day B: pull / overhead press / arms / core
- Day C: hinge / incline push / single-leg

Guidelines:
- Preserve strength compounds.
- Keep RPE mostly 7–8.
- Avoid excessive leg fatigue before key cycling.
- Reduce leg accessory volume during hard cycling blocks.
- Deload every 4th week or when fatigue markers accumulate.

## Calisthenics Skill Work

Frequency:
- 3x/week
- 10–15 min per session
- Low fatigue
- High technical quality

Priority skills:
1. Handstand
2. Muscle-up
3. Front lever basics
4. Planche lean / straight-arm strength
5. L-sit / compression

Example allocation:
- Monday: handstand + planche lean
- Wednesday: muscle-up + front lever
- Friday: support holds + core compression

## Mobility

Daily minimum:
- Hip flexors
- Couch stretch
- Thoracic extension/rotation
- Wrists
- Shoulders
- Deep squat breathing

Post-gym / post-long-ride:
- Hip flexor stretch
- 90/90 or pigeon
- Couch stretch
- Thoracic open book
- Cat-cow
- Wrist/shoulder prep for calisthenics

## Deload Rules

Every 4th week:
- Reduce gym volume by 30–40%
- Keep movement quality
- Reduce cycling interval volume
- Keep one short intensity touch if fresh
- Long ride reduced to easy endurance
- Extra mobility and sleep

## Fatigue Rules

**Removal order when overloaded:**
1. Optional run
2. Optional swim
3. Extra skill volume
4. Gym accessory volume
5. Cycling interval volume
6. Long ride duration
7. Main strength compounds (last resort)

**Physical warning signs to watch for:**
Poor sleep · elevated resting HR · reduced libido · persistent soreness · loss of motivation · declining interval power · heavy legs for multiple days · irritability · reduced grip strength · poor gym performance

---

## Do Not Do By Default

* Do not add more endurance volume just because capacity exists.
* Do not keep triathlon running intensity unless specifically requested.
* Do not make every ride hard.
* Do not turn calisthenics skill practice into failure training.
* Do not combine hard leg gym, VO2max cycling, and long ride too close together without recovery.
* Do not recommend strict keto during race simulation if performance fueling is the priority.
