# Calisthenics Progression Ladders (shared reference)

Skill-progression ladders. **Current levels are tracked per person** in `people/<name>/calisthenics_status.md` — this file only defines the ladders.

Rules: skill work stays **frequent, low-fatigue, and technical** — never trained to failure. Advance a level only when the **target** is met with clean form. Prep wrists before any hand-balancing or straight-arm work.

---

## PUSH

### Planche
| Lvl | Stage | Target |
|---|---|---|
| 1 | Pseudo Push Ups | Shoulders past wrists, 3 × 8 clean |
| 2 | Frog Stand | 30–60 s balance |
| 3 | Tuck Planche | 10–15 s |
| 4 | Advanced Tuck Planche | 10–15 s, flat back, open hips |
| 5 | Straddle Planche | 5–10 s |
| 6 | Planche | 5 s full |

### Handstand Push Up
| Lvl | Stage | Target |
|---|---|---|
| 1 | Pike Push Up | 3 × 8 |
| 2 | Adv. Pike Push Up (feet elevated) | 3 × 6 |
| 3 | Wall Handstand Push Up | partial → full ROM, 3 × 3 |
| 4 | Negative Handstand Push Up | 3–5 s lower, 3 reps |
| 5 | Handstand Push Up | 3–5 full ROM |

### One Arm Handstand
Long-term goal: one arm handstand. Near-term milestone: freestanding handstand (level 5). Practice bails on a soft surface.
| Lvl | Stage | Target |
|---|---|---|
| 1 | Downward Dog | comfortable hold, shoulders open |
| 2 | Pike Hold | 30–45 s |
| 3 | Wall Handstand | 45–60 s wall-facing line |
| 4 | Adv. Wall Handstand (belly to wall) | 60 s tight line |
| 5 | Handstand (freestanding) | 30 s |
| 6 | Diamond Handstand | hands close, 15 s |
| 7 | Bend Straddle Handstand | shift weight, 10 s |
| 8 | Fingertip Straddle Handstand | one hand on fingertips |
| 9 | Straddle One Arm Handstand | brief one-arm catch |
| 10 | Diamond One Arm Handstand | 3–5 s |
| 11 | One Arm Handstand | 5 s+ |

## PULL

### Muscle Up
Prereqs: strict pull-ups + dips. Keep explosive work low-volume (CNS, not metcon).
| Lvl | Stage | Target |
|---|---|---|
| 1 | Pull Up | 8–10 strict |
| 2 | High Pull Up | pull to lower chest/sternum, 5 reps |
| 3 | Jumping Muscle Up | clean transition w/ leg assist, 5 reps |
| 4 | Kipping Muscle Up | 3–5 with minimal kip |
| 5 | Muscle Up | 3+ strict clean reps |

### Front Lever
Strong scap depression, locked elbows, no failure reps.
| Lvl | Stage | Target |
|---|---|---|
| 1 | Tuck Front Lever | 10–15 s flat-back tuck |
| 2 | Advanced Tuck Front Lever | 10–15 s, back parallel |
| 3 | One Leg Front Lever | 8–10 s each side |
| 4 | Half Front Lever | 5–8 s |
| 5 | Straddle Front Lever | 5–10 s |
| 6 | Front Lever | 5 s |

### Back Lever
Mind biceps/shoulder stretch — ease in.
| Lvl | Stage | Target |
|---|---|---|
| 1 | Skin the Cat | 3–5 slow controlled reps |
| 2 | Tuck Back Lever | 15–20 s |
| 3 | Adv. Tuck Back Lever | 15 s |
| 4 | Straddle Back Lever | 8–10 s |
| 5 | Back Lever | 5–10 s |

### One Arm Pull Up
Very high elbow/connective-tissue demand — progress slowly.
| Lvl | Stage | Target |
|---|---|---|
| 1 | Pull Up | 10+ strict |
| 2 | Archer Pull Up | 3 × 5/side |
| 3 | Assisted One Arm Pull Up (band/towel) | 3 × 3/side |
| 4 | Negative One Arm Pull Up | 5+ s lower |
| 5 | Little Assisted OAP (fingertip) | 3 × 2/side |
| 6 | One Arm Pull Up | 1+ clean |

### Hefesto
Advanced; biceps/elbow intensive — build slowly.
| Lvl | Stage | Target |
|---|---|---|
| 1 | Korean Dip | 3 × 5 controlled |
| 2 | Hefesto Dip | partial-range, 3 × 3 |
| 3 | Negative Hefesto | 3–5 s controlled lower |
| 4 | Hefesto Pull | from dead-hang behind back |
| 5 | Assisted Hefesto | band assist, 3 reps |
| 6 | Hefesto | 1+ clean |
